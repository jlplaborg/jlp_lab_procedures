# Usage

`$> ls [$directory] [-l] [-a] [-h] [-S] [-1]`

---
# Description

**List**. Display all contents of a directory to the terminal screen.

---

# Important Options

## $directory

The directory to list. If no `$directory` is specified, then the current working directory is used instead

## -l

**Long**. Display detailed information about each file

## -a

Display hidden files alongside visible files

## -h 

**Human-readable**. Display the size of each file in human-readable format

## -S

**Sort**. Display files in order of size

## -1

**One**. Display only one line per file
