# Usage

`$> cd [$destination | -]`

---
# Description

**Change directory**. This command moves your terminal location to a new location. If no arguments are provided, you are moved to the `home` or `~` directory

---

# Important Options

## $destination

A directory path to move to. This can be a relative or an absolute path. Remember that `..`, `~`, and `/` are valid inputs as a destination.

## -
Move to the previous location. Think of the `-` argument as essentially being the "back" button in your terminal
