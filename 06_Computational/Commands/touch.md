# Usage

`$> touch $filename [-c, --no-create]`

---
# Description

Touch a file.  If a file exist, it will update the "last updated" timestamp to be the current time. If a file does not exist, it will create an empty file with the provided name

---

# Important Options

## $filename

The name of the file to touch

## -c, --no-create

This option will prevent the creation of a file if it does not exist.
