# Cleavage
## Summary
This is the last step in the process for collecting data for a peptide. This will give us the thermodynamic data that is needed to make evaluations on helicity, melting temperatures, and ΔΔG's.
## Materials
- 1 mm Glass cuvettes (can use up to 6)
- Blank CD solution
- Peptide CD solution
- Kim wipes
- Vacuum cuvette cleaner 
- Water
- Acetone
## Reagents
- Water (MQ)
- Acetone
- [[Guanidinium chloride]]
- Peptide
- [[Phosphate Buffer]]
## Procedure
Workflow: Turning on the machine
1. Turn on the Nitrogen - let purge for 5 minutes
2. Turn on instrument
3. Turn on coolant system
4. Turn on computer 
Workflow: Opening Programs
5. Go to spectra manager app
	1. can click out of nitrogen purge count down
6. For wavelength scans:
	1. Open "spectra manager" by:
		1. Clicking "Applications"
		2. Click "Start"
		3. Open Parameters
			1. "wavelength_scan" --> Open
		4. Change anything on the parameters that we need for saving purposes (i.e. folder, sample names, comments, dates, etc)
			1. If you save it as Sample-No it will ask you to have information in the text boxes for each same. If you aren't running them all, just fill them in with N/A and an option will come up after you hit sample to select cells
		5. Open the shutter
		6. Click "Sample"
7. For Melting scans:
	1. Open "Temperature/" by:
		1. Clicking "Applications"
		2. Click "Start"
		3. Open Parameters
			1. Pick whatever parameters file you want
		4. Change anything on the parameters that we need for saving purposes (i.e. folder, sample names, comments, dates, etc)
			1. If you save it as Sample-No it will ask you to have information in the text boxes for each same. If you aren't running them all, just fill them in with N/A and an option will come up after you hit sample to select cells
		5. Click "Sample"
Workflow: Turning off the Machine
8. Save any data on a flash drive
9. Turn off the lamp (wait to turn off machine/nitrogen for 5-10 minutes to allow the lamp to cool)
10. Close out of the program
11. Turn off CD instrument once lamp is cool
12. Turn off coolant system
	1. Make sure the coolant system is equilibrated
13. Turn off nitrogen flow
